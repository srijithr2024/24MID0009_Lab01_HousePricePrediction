## Project Overview
This project predicts house prices using supervised machine learning techniques. Two datasets were used:
1. California Housing Dataset
2. Ames Housing Dataset

The project includes:
1. Data loading and preprocessing
2. Exploratory Data Analysis (EDA)
3. Simple and Multiple Linear Regression
4. Ridge Regression
5. Elastic Net
6. Random Forest
7. Gradient Boosting
8. Cross-validation
9. Hyperparameter tuning using GridSearchCV
10. Model evaluation using MAE,RMSE,R2_score,andAdjusted R2_score
